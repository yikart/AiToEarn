# User Store

用户登录态与账户余额相关全局状态目录。该目录只服务 `useUserStore`，其中 `utils/` 是用户 store 私有工具，不允许其他模块直接复用；如出现跨模块复用需求，必须先迁移到中立公共目录并更新 README。

## 文件清单

| 文件                  | 说明                                                                                                 |
| --------------------- | ---------------------------------------------------------------------------------------------------- |
| `index.ts`            | `useUserStore` 主入口，管理登录态、用户信息、Credits、广告主余额、创作者可提现余额、区域切换提示等。 |
| `utils/device.ts`     | 用户每日访问上报使用的客户端设备、系统、浏览器识别。                                                 |
| `utils/ipLocation.ts` | 用户 store 使用的 IP 地理位置获取、国家提取和国家代码解析。                                          |

## 使用规则

- 外部只允许通过 `@/store/user` 引用 `useUserStore` 和公开类型。
- `utils/` 内文件默认是 `useUserStore` 私有实现，其他页面、组件、store 禁止直接引用。
- 如果 `utils/` 内能力需要跨模块复用，先迁移到 `src/utils` 或合适业务公共目录，并同步更新对应 README。
