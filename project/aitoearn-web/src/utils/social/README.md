# Social 工具

社交账号共享工具目录，放跨页面复用的账号统计和账号展示辅助逻辑。

## 文件清单

| 文件         | 导出                                                      | 说明                                   |
| ------------ | --------------------------------------------------------- | -------------------------------------- |
| `account.ts` | `calcSocialAccountsTotalFans`、`shouldShowFansSyncNotice` | 社交账号粉丝数汇总与粉丝同步提示判断。 |

## 使用规则

- 只放跨页面复用的社交账号纯工具。
- 渠道授权、发布流程等平台交互逻辑应放 `src/api/channels/`、`src/store/plugin/` 或对应业务目录。
