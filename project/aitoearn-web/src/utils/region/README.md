# Region 工具

区域与域名切换工具目录，处理国内/国际版本域名切换和区域限制提示。

## 文件清单

| 文件          | 导出                            | 说明                                 |
| ------------- | ------------------------------- | ------------------------------------ |
| `domain.ts`   | `switchRegionDomain`            | 根据当前区域获取目标域名并执行跳转。 |
| `redirect.ts` | `confirmPlatformRegionRedirect` | 平台区域限制确认弹窗与跳转。         |

## 使用规则

- 只放区域版本、域名切换、区域限制提示相关逻辑。
- 单个 store 私有的区域判断应下沉到对应 store。
