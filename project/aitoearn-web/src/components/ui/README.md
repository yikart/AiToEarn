# UI Components

本目录存放 shadcn/ui 基础组件与项目级 UI primitive，只放低业务语义的按钮、弹窗、表单、选择器等基础能力。

## 组件清单

| 组件                    | 说明                                                                                                                                                                                      |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `searchable-select.tsx` | 可搜索选择器（基于 command + popover，选项支持 `icon` / `color` / `description` 展示；支持本地过滤或 `searchValue` / `onSearchChange` 远程搜索，可通过 `loadingMode` 控制加载态展示位置） |
