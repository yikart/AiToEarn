# ImageStack

`AiBatchGenerateBar` 的私有媒体堆叠组件，负责本地图片、视频、音频的折叠展示、展开交互、删除动画、上传入口和媒体预览。

## 目录结构

| 目录          | 说明                                                      |
| ------------- | --------------------------------------------------------- |
| `index.tsx`   | 入口组合层，只连接 hooks、移动端/桌面端容器和媒体预览。   |
| `components/` | 当前堆叠组件私有 UI 子组件，例如媒体卡片、进度层和容器。  |
| `hooks/`      | 当前堆叠组件私有 Hook，例如响应式、展开、删除和视频信息。 |
| `types/`      | 当前堆叠组件私有 props、回调、布局状态和视频信息类型。    |
| `utils/`      | 当前堆叠组件私有常量、媒体展示名称、预览类型和样式计算。  |

## 维护规则

- `index.tsx` 只保留组合逻辑，不再直接追加渲染分支、事件处理、类型或常量。
- 新增媒体展示区块放入 `components/`，不要把子组件写在入口文件里。
- 新增交互状态放入 `hooks/`，例如手势、hover、动画计时器和媒体信息读取。
- 新增纯计算放入 `utils/`，例如旋转角度、布局样式和媒体类型转换。
- 新增或调整 props、视图模型、回调类型时统一放入 `types/`。
- 如果后续多个 `AiBatchGenerateBar` 子组件需要共享这些媒体状态，再评估迁入上层局部 `store/`。
