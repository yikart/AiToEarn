# ToolBarInline

`AiBatchGenerateBar` 的私有内联工具栏，负责组合生成模式、平台、模型、参数、比例、时长、数量、快捷链接和提交控件。

## 目录结构

| 目录          | 说明                                                |
| ------------- | --------------------------------------------------- |
| `index.tsx`   | 入口组合层，只连接 props、翻译、hooks 和子组件。    |
| `components/` | 当前工具栏私有 UI 子组件，每个组件一个独立文件夹。  |
| `hooks/`      | 当前工具栏私有 Hook，例如弹层状态、派生数据和事件。 |
| `utils/`      | 当前工具栏私有纯函数和样式 class 常量。             |
| `types/`      | 当前工具栏私有 props、视图模型和辅助类型。          |

## 维护规则

- `index.tsx` 不再直接堆积大段 JSX、业务计算或事件处理。
- 子组件只接收自身渲染所需 props，不反向 import 父组件实现细节。
- Hook 放入 `hooks/`，类型放入 `types/`，纯函数和常量放入 `utils/`，不要混放到同一文件。
- 如果工具栏内部出现跨多个子组件共享的可写状态，再在上层 `AiBatchGenerateBar/store/` 建立局部 store。
