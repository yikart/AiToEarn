# Chat Components

`Chat/` 存放聊天和 AI 任务会话相关共享组件。

## 组件清单

| 组件/目录             | 说明                                   |
| --------------------- | -------------------------------------- |
| `ActionCard`          | 聊天动作卡片。                         |
| `ChatInput`           | 聊天输入组件。                         |
| `ChatMessage`         | 聊天消息展示组件。                     |
| `MediaUpload`         | 媒体上传组件。                         |
| `Rating`              | 评分组件。                             |
| `Share`               | 聊天任务分享弹窗、预览和长图生成。     |
| `TaskCard`            | 聊天任务卡片。                         |
| `TaskHistoryList.tsx` | 任务历史列表。                         |
| `utils`               | 聊天消息转换、内容解析和任务状态工具。 |

## 放置规则

- 聊天任务、消息、分享、会话工具放在这里。
- 如果聊天页局部工具被全局组件复用，应提升到 `Chat/utils`，不要让全局组件 import 页面私有 `src/app/[lng]/chat/...`。
