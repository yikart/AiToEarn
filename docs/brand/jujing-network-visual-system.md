# 巨鲸网络视觉品牌系统

## 品牌核心

巨鲸网络是一家面向 AI 智能体网络、自动化获客、内容增长和企业级增长基础设施的新锐科技公司。

最终品牌方向调整为明亮、开放、国际化的高级科技公司气质，参考世界级科技企业的长期主义品牌表达：干净留白、清爽颜色、亲和但可信，不走暗黑科幻和紫色 AI 模板。

一句话定位：

> 巨鲸网络，用 AI 智能体网络连接内容、客户和增长。

英文辅助语：

> Build intelligent connections, drive real growth.

## Logo 概念

最终标志采用抽象几何波形，而不是具象鲸鱼。

符号含义：

- `W / J` 负空间：呼应 Jujing Network 和网络波形。
- 宽阔流动弧线：表达“巨鲸”的规模感和稳定推进感。
- 蓝青渐变：表达 AI 智能流动和科技可信度。
- 绿色节点：表达连接、增长、客户触达。
- 小尺寸可识别：适合 App icon、浏览器插件、favicon、控制台顶部栏。

## 色彩系统

| Token | Hex | 用途 |
| --- | --- | --- |
| Cloud White | `#ffffff` | 主背景、App 图标底色、品牌留白 |
| Mist Gray | `#f2f4f7` | 页面底色、浅色分区 |
| Ink | `#102033` | 标题、正文、单色 Logo |
| Whale Blue | `#2563eb` | 主品牌色、主按钮、关键图形 |
| Signal Cyan | `#00bbd9` | AI 信号、连接线、数据高亮 |
| Growth Green | `#22c55e` | 增长节点、成功态、客户连接 |
| Warm Spark | `#f59e0b` | 少量提示、亮点、活动状态 |

色彩比例：

- 70% Cloud White / Mist Gray
- 18% Ink / Graphite 文本结构
- 10% Whale Blue / Signal Cyan
- 2% Growth Green / Warm Spark

## 字体与排版

中文品牌名使用现代几何黑体气质，保持清晰、宽松、稳定，避免过度装饰。英文 `JUJING NETWORK` 使用干净国际化无衬线，字距略放开，作为辅助标识。

排版原则：

- 留白多，信息层级清楚。
- 标题短而有力量。
- 不用空泛口号。
- 产品界面保持工具属性，品牌感通过 Logo、色彩和状态反馈表达。

## 产品 UI 落地规范

- 侧栏与移动顶栏使用白底圆角 App icon 版本。
- 分享图、短链页和插件图标使用同一套蓝青绿标志。
- 控制台主界面保持轻量，不做暗黑重科技。
- AI 执行、连接、增长状态优先使用 Signal Cyan / Growth Green。

## 不做什么

- 不使用暗黑科幻主调。
- 不使用大面积紫色渐变。
- 不画具象鲸鱼、卡通鲸鱼或机器人。
- 不使用复杂发光插画。
- 不把后台做成营销页面。
